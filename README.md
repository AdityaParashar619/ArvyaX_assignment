# AI-Assisted Journal System

## Overview

The AI-Assisted Journal System allows users to write journal entries after immersive nature sessions such as forest, ocean, or mountain environments.

The system stores journal entries, analyzes the emotional sentiment of each entry, and provides insights about the user's mental state over time.

This project demonstrates API design, backend architecture, and integration of an AI analysis layer.

---

## Features

- Create journal entries
- Store entries in a database
- Analyze emotional sentiment of journal text
- Retrieve past journal entries
- Generate insights from user activity

---

## Tech Stack

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Frontend
- React

### AI / NLP Layer
- Sentiment analysis using a lightweight NLP library  
  (architecture supports replacement with LLM services such as Gemini or OpenAI)

---

Insights include total entries, most used ambience, and top detected emotion.
